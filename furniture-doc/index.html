    Installation

    Getting started

    Download our template from marketplace

    Unzip it in your desired locatione

    Open terminal in that directory

    Run below commands:

    # It will install all packages
npm install

    Generating build for production

    To generate a production build, use the following commands:

    # It will build your project
npm run build

    Once the build is generated, you can preview it on your local system using the following commands:

    # It will run your project
npm run dev

    Layout Folder

    layouts

    Our template comes with layouts contain <header> & <footer>
    You can find it in src/components/ directory

    <Header>

    file name Header.js
    Header html used in layout.js file with initialvalues_data

    <Footer>

    file name Footer.js
    Footer html used in layout.js file with initialvalues_data

    @component

    Place your component in this directory. Components in this dir will use for carry all Html
    You can find it in src/components directory

    Section

    Section folder contain all html code which use in project
    You can find it in src/components/Sections directory

    @Pages

    Contains your app pages. Just create a new file and new route will be auto generated
    based on file name and its placement in the app directory.
    You can find it in src/app directory

    Data

    In data folder we make the json dummy data.
    You can find it in public/json/data directory

    Custom Fetching

    ● We provide custom solution from your data fetching needs. In most of the case,
    we generally configure the fetching configuration like adding base URL for all of our API calls,setting headers & footer.

    // No need to repeat `https://localhost/api` now
const { data } = await UseFetchData('/users')

    useFetchData() function in src/hook/usegetdata.js

    Public

    @public

    It contains logos and other static assets. You can place your static assets here.
    You can find it in project/public directory

    Writing your own CSS
    ● You can write your own CSS in src/app/globals.css file.

    Project contain coustom css
    ● Project contain coustom css in public/css folder.

    Project contain coustom images
    ● Project contain coustom css in public/image folder.

    Theme Config

    Project contain all api or data static path or url in themeconfig file.
    You can change your api url with the static dummy data url.
    You can find it in src/config/themeConfig directory
